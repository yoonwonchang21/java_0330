웹 프로그래밍

[기초 세팅]
1. 데이터베이스 생성 → 테이블 생성
2. 웹 프로젝트 생성
3. (기능구현에)필요한 라이브러리 추가  
	- 데이터베이스 JDBC(ojdbc6)
	- Connection Pool(DBCP)
	- file upload관련
	- JSTL
4. MVC패턴 구현
	- Front Controller(=Servlet)
	- Command 패턴 → Service(I), 
	- DAO 패턴 → DAO Classes
	- View JSP → EL, JSTL
5. DBCP Pool 등록 - Servlet

→ When all ready, Server Start!!

2. DAO에 기능(메서드)추가
3. View 생성 → JSP로
4. 설정 파일에 사용자 요청 URI와 실행 클래스를 등록
